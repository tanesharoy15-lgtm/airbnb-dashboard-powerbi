# Airbnb Dashboard Analysis

## 📊 Project Overview
This project presents an interactive Power BI dashboard analyzing Airbnb listings data. It provides insights into pricing, availability, and location-based trends.

## 🎯 Objectives
- Analyze pricing trends across locations
- Identify high-demand areas
- Understand listing distribution and availability

## 🛠 Tools Used
- Power BI
- Excel / CSV Dataset
